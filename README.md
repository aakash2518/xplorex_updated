# Xplorex - Premium Travel Partner

Your premium travel partner for handcrafted experiences across 12+ countries.

## Overview
Xplorex is a modern, interactive web application built to offer premium travel experiences. It features immersive 3D graphics and fluid animations to provide a stunning user interface for exploring travel destinations and packages.

## Tech Stack
- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI / shadcn-ui
- **3D Rendering**: Three.js, React Three Fiber, React Three Drei
- **Animations**: Framer Motion
- **Forms & Validation**: React Hook Form, Zod
- **Icons**: Lucide React

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Set up environment variables**:
   Copy `.env.example` to `.env` and configure your environment-specific variables.
   ```bash
   cp .env.example .env
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.

## Available Scripts
- `npm run dev` - Starts the development server.
- `npm run dev:secure` - Starts the dev server with experimental HTTPS.
- `npm run build` - Builds the application for production.
- `npm run start` - Starts the production server.
- `npm run lint` - Lints the codebase.
